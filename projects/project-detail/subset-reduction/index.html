Proof of Hardness: Subset-Sum Problem



Donghyuk Kim, Richard Yang, Alex Xu, Seunghyun Nam
Computability and Complexity CSCI3390 Project, Fall 2018
Report, Github

The paper proves that the classic SUBSET-SUM problem is an NP-COMPLETE problem. If there exists an algorithm that can solve the SUBSET-SUM in polynomial time, P would equal NP.
The github program executes a polynomial time reduction from 3-SAT --> SUBSET-SUM, then solves SUBSET-SUM in pseudo-polynomial time. 3-SAT is a known NP-Complete problem. The reduction shows that SUBSET-SUM is also NP-Complete. If we find a polynomial time algorithm to solve SUBSET-SUM, we would be able to solve 3-SAT in polynomial time and prove P = NP.
