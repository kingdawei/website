Script Analysis of Crime Show



Donghyuk Kim, Ashley Oh, Jessica Bolduc, Dipesh Patel
Natural Language Processing CSCI3349 Project, Fall 2019
Download-Poster

Can we differentiate murderers and non-murderers from the TV show "Crime Scene Investigation" by analyzing the scripts? We utilized Natural Language Processing techniques such as sentiment analysis, affective, and machine learning to classify characters into murderers and non-murderers.
