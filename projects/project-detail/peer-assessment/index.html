Peer Assessment System



Donghyuk Kim, Richard Yang, Alex Xu, Seunghyun Nam
Software Engineering CSCI3356 Project, Spring 2020
Report, Github, Application

Full-stack peer assessment management system developed by a team of four seniors. This semester-long project was designed to emulate real software engineering process. Professors can create and manage peer evaluations for their classes and students can complete assigned peer evaluations. Notable features include: automatic email reminders, downloadable statistics, etc. See the report for details on all functional requirements that have been fulfilled.
